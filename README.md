# The RISC-V and Open Source Hardware/Software Ecosystem

This is a comprehensive, curated list of essential tools, repositories, and organizations dedicated to advancing the **RISC-V** Instruction Set Architecture (ISA) and the broader **Open Source Hardware** and **Software** landscape.

---

## Official RISC-V Initiatives

* **RISC-V International** - The primary organization maintaining and standardizing the RISC-V ISA.
    * [https://riscv.org/](https://riscv.org/)
    * **RISC-V Training**: Access official training and certification programs.
        * [https://riscv.org/community/training/](https://riscv.org/community/training/)
    * **"Learn RISC-V"**: A fantastic community-driven GitHub repository offering a massive list of learning resources.
        * [https://github.com/riscv/learn](https://github.com/riscv/learn)
    * **RISC-V Ecosystem Landscape**: Explore the complete landscape of all RISC-V projects, products, and contributing organizations.
        * [https://landscape.riscv.org/](https://landscape.riscv.org/)
    * **RISC-V Developer Boards**: Information on how to acquire free RV32/RV64 development boards for testing and projects.
        * [https://riscv.org/developers/boards/](https://riscv.org/developers/boards/)

## General Open Source

### For CPU and Silicon Development (EDA)

* **Yosys**: The foundational open source framework for **Verilog RTL synthesis**.
    * [https://github.com/YosysHQ/yosys](https://github.com/YosysHQ/yosys)
* **OpenROAD Project**: Provides a fully automated and open source flow for generating physical chip layouts (RTL-to-GDS).
    * [https://github.com/The-OpenROAD-Project](https://github.com/The-OpenROAD-Project)
* **KLayout**: A widely-used open source EDA tool primarily for **silicon layout viewing** and editing.
    * [https://www.klayout.de/](https://www.klayout.de/)
* **OpenXC7**: A free and open source FPGA toolchain specifically for AMD/Xilinx Series 7 chips.
    * [https://github.com/openxc7](https://github.com/openxc7)
* **zynq-mkbootimage**: An open source replacement for the proprietary Xilinx `bootgen` application.
    * [https://github.com/antmicro/zynq-mkbootimage](https://github.com/antmicro/zynq-mkbootimage)

### For Embedded Programming

* **Zephyr Project**: A highly scalable open source **Real-Time Operating System (RTOS)** supporting over 750 embedded boards.
    * [https://www.zephyrproject.org/](https://www.zephyrproject.org/)
* **Yocto Project**: The Linux Foundation project that enables developers to create **custom, tailored Linux distributions** for embedded devices.
    * [https://www.yoctoproject.org/](https://www.yoctoproject.org/)
* **VS Code (Visual Studio Code)**: A powerful, extensible open source code editor widely used for cross-platform development.
    * [https://code.visualstudio.com/](https://code.visualstudio.com/)

### For Circuit Design

* **KiCad**: The leading open source EDA suite for comprehensive **PCB design** and schematic creation.
    * [https://www.kicad.org/](https://www.kicad.org/)
* **Qucs-S**: An open source circuit simulation software that is fully compatible with popular SPICE models like LTSpice.
    * [https://ra3xdh.github.io/](https://ra3xdh.github.io/)

## Community and Advocacy Organizations

Non-profits and foundations that work on broad open source software and hardware development globally.

* **Linux Foundation**: Supports Linux and numerous large-scale open source projects through governance and resources.
    * [https://www.linuxfoundation.org/](https://www.linuxfoundation.org/)
* **GNU Project**: The original free software initiative that maintains the essential GNU toolchain and defines the concept of **Free Software**.
    * [https://www.gnu.org](https://www.gnu.org)
* **Open Source Hardware Association (OSHWA)**: Certifies open source hardware designs and promotes the free exchange of physical designs.
    * [https://oshwa.org/](https://oshwa.org/)
* **Free Open Source Silicon Foundation (FOSSi Foundation)**: A non-profit dedicated to fostering the complete open sourcing of semiconductor technology.
    * [https://fossi-foundation.org/](https://fossi-foundation.org/)
* **Open Source Firmware Foundation**: Supports the development and adoption of open source firmware solutions (e.g., BIOS/UEFI replacements).
    * [https://opensourcefirmware.foundation/](https://opensourcefirmware.foundation/)
